```shell
#屏幕向下移动一页
ctrl+f
#屏幕向上移动一页
ctrl+b

#移动到这一行的最前面字符处
0
#移动到这一行的最后面字符处
$
#移动到这个档案的最后一行
G
#移动到这个档案的第一行
gg

#向光标之下寻找一个名称为word的字符串
/word
#向光标之上寻找一个名称为word的字符串
?word
#重复前一个搜索的动作
n
#第n1与n2行之间寻找word1这个字符串,并将该字符串取代为word2
:n1,n2s/word1/word2/g
#第一行到最后一行寻找word1字符串,并将该字符串取代为word2
:1,$s/word1/word2/g

#删除光标所在的行
dd
#复制光标所在的行
yy
#p将已复制的数据在光标下一行贴上,P将已复制的数据在光标上一行贴上
p,P
#复原前一个动作
u
```
