**scanState:表示WorkQueue的扫描状态**

**int类型:**

|  **位**   | **描述**                                                     |
| :-------: | :----------------------------------------------------------- |
|  **32**   | **表示WorkQueue的扫描状态,等于1表示非扫描状态**              |
| **31-17** | **表示scanState的更新记录,每次更新,都会自增1<br/>二进制位是从17位开始,所以每次增加SS_SEQ(1<<16)** |
| **16-1**  | **WorkQueue在WorkQueue的数组下标,是一个奇数<br/>最低位等于0时表示WorkQueue对应线程获取到了一个任务,处于执行任务的状态** |
