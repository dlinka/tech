#### print

```py
print("你好，新世界", end="->") # end结束符
print("你好", "新世界", sep="，") # sep分隔符
print("你好" + "新世界") # 连接字符串
print("你好，新世界" * 3) # 复制3遍
print(r"c:\n") # r后面字符串全部转义
print('你好 "新世界"') # 单引号双引号
print(f"i = {i++}")
```
